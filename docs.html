Raccourcis clavier

² ou ~ ou @
Ouvrir / fermer le terminal CLI. La touche ² existe sur AZERTY, ~ et @ sur QWERTY.
Esc
Fermer le CLI, l'explorateur de bloc, le mode rayons-X, ou tout modal ouvert.
↑ ↑ ↓ ↓ ← → ← → B A
Code Konami → active / désactive le mode rayons-X (dump JSON de l'état interne en direct).
Tab
Parcours au clavier de tous les éléments interactifs. Focus ring cyan visible.
Entrée / Espace
Active le contrôle focalisé (boutons, wallet HUD, nœuds consensus…).
Souris — drag
Faire tourner le tesseract (3D).
Shift + drag
Tourner le tesseract sur son axe W (4ᵉ dimension).

Commandes CLI

Moteur Explorer

help
Affiche la liste complète.
mine
Scroll vers la section Mining et lance le minage.
tamper
Scroll vers la section Tamper et altère le bloc #001 automatiquement.
reset
Réinitialise la chaîne de démonstration.
fly <section>
Navigue vers hero, concept, dim, time, sim, tamper, mine, outro.
audio
Bascule l'audio génératif (Tone.js).
nodes
Affiche l'état réseau simulé.
hash <texte>
Calcule le SHA-256 d'une chaîne via Web Crypto API.
pulse
Fait pulser le tesseract manuellement.
red
Bascule en mode alerte (éphémère).
clear / exit
Efface l'historique du terminal / ferme.

Extensions v2.1 (wallet, VM, consensus)

wallet
Infos détaillées de votre wallet.
balance
Solde TSC courant.
stake <n>
Mise n TSC — simulation PoS, restitué après 4 s.
deploy counter
Déploie un contrat compteur.
deploy vote
Déploie un contrat de vote binaire.
call <addr> <method> [args…]
Appelle une méthode d'un contrat déployé.
contracts
Liste les contrats déployés cette session.
mempool
Dump des transactions en attente.
explore <n>
Ouvre l'explorateur de bloc sur le bloc #n de votre leaderboard.
byzantine on|off
Simule ~46 nœuds malveillants (passe online à 210/256).
xray
Bascule le mode rayons-X.

API JavaScript exposée

Ouvre la console du navigateur (F12) et joue avec ces objets globaux :

window.TSC                // Store interne v2.1 (wallet, chain, mempool, …)
window.TSCCore            // Moteur explorer — difficulté, horloge
window.TSCCore.setDifficulty(n)  // 2 ≤ n ≤ 6
window.TSCCore.getDifficulty()   // '0000'
window.TSCUi              // Primitives UI
window.TSCUi.toast(msg, { level, duration })
window.TSCUi.modal({ title, body, actions })
window.TSCUi.safeSetItem(key, value)
window.TSCAcademy         // Progression académie
window.TSCAcademy.state   // Snapshot figé
window.TSCAcademy.reset()
window.TSCAcademy.markChapterDone(n)
window.TSCAcademy.glossary.show(term, anchor)
window.TSCLifecycle       // Registre timers
window.TSCLifecycle.intervals  // Set<id>
window.TSCLifecycle.cleanup()  // Tout annuler manuellement

Easter eggs

  • Code Konami → mode rayons-X plein écran. Les initiés savent pourquoi ↑↑↓↓←→←→BA.
  • Clic sur "SOMMETS ILLUMINÉS" en haut-gauche → récap complet de votre activité de minage.
  • Clic sur un bloc du leaderboard de minage → panneau explorer slide-in avec merkle root visualisé en arbre binaire.
  • Clic sur le wallet HUD → modal détail avec bouton « copier l'adresse ».
  • Glossaire flottant → cliquez n'importe quel mot souligné cyan dans le corps du texte.

Stack technique

Rendu 3D/4D
Three.js r128 (CDN avec SRI)
Audio
Tone.js 14.8.49 (CDN avec SRI)
Cryptographie
Web Crypto API native (SHA-256, ECDSA P-256)
Stockage
localStorage (wallet, leaderboard, progression, onboarding)
Style
CSS pur, variables CSS, backdrop-filter avec @supports
Typographie
Impact + SF Mono + system fonts (zéro @font-face)
Build
Aucun. Pas de transpiler, pas de bundler, pas de npm install.
Déploiement
GitHub Pages auto via Actions (workflow deploy.yml)
