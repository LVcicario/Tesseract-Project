TEST D'INTÉGRITÉ // 005

Modifiez un bloc.
Regardez la chaîne se briser.

Chaque bloc contient le hash du précédent. Altérez une donnée, son hash change — tous les blocs suivants deviennent invalides. Avertissement : le tesseract va réagir.

Modifiez le montant du bloc #001 (input jaune) — observez la cascade d'invalidation en temps réel. Le réseau rejetterait cette chaîne en <50ms.

▸ Effet domino détecté. Le bloc modifié a changé de hash. Comme chaque bloc référence celui qui le précède, tous les suivants se retrouvent avec un hash-parent invalide. Aucun nœud honnête n'acceptera cette chaîne : elle est orpheline. Coût pour un attaquant : re-miner tous les blocs postérieurs plus vite que le réseau honnête — exponentiellement improbable.
