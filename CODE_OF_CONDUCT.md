# Code de conduite

TESSERACT.CHAIN est un projet ouvert, pédagogique, qui s'adresse notamment à des apprenants. Tout le monde doit pouvoir y contribuer, y lire du code, y poser des questions sans crainte.

## Principes

**On est bienveillants.** Une question « bête » n'existe pas. Une remarque qui met l'auteur mal à l'aise ne fait avancer personne.

**On est rigoureux.** Le code doit être correct, l'explication doit être vraie. Corriger une erreur technique est toujours le bienvenu ; le faire avec mépris ne l'est jamais.

**On est patients.** Les chapitres de l'Académie sont pensés pour des débutants. Les retours doivent aider à simplifier, pas à complexifier.

## Attendus

- Être respectueux envers les autres contributeurs et contributrices, quels que soient leur niveau, leur origine, leur parcours.
- Accepter les critiques constructives ; en formuler sans attaque personnelle.
- Préférer les exemples et les contre-exemples aux jugements.
- Reconnaître ses erreurs et ajuster — c'est la même démarche qu'apprendre la blockchain.

## Inacceptable

- Harcèlement, propos discriminatoires, intimidation.
- Attaque personnelle plutôt que débat technique.
- Partage d'informations privées sans consentement.
- Toute autre conduite contraire à l'esprit éducatif et bienveillant du projet.

## Application

Si vous voyez un comportement inapproprié, ouvrez une issue privée auprès du mainteneur (actuellement [@LVcicario](https://github.com/LVcicario)) ou signalez-le via les outils GitHub standards.

Les mainteneurs se réservent le droit de modérer, retirer, refuser ou bannir toute contribution ou contributeur jugé incompatible avec ce code.

Ce document s'inspire du [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) adapté à l'esprit du projet.
