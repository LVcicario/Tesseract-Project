CHAPITRE 01 · ★ DÉBUTANT
Le hash

Une fonction qui transforme n'importe quel texte en une empreinte unique de 256 bits. Déterministe, sensible, irréversible. La brique de tout ce qui suit.

Le bloc

Pourquoi chaîner les blocs par leur hash rend la falsification pratiquement impossible. L'immuabilité, expliquée sans incantation.

Les identités

PROMESSES // CE QUE CE PARCOURS EST ET N'EST PAS
✓ Ce que c'est
  • Un parcours pour comprendre, pas pour spéculer.
  • Du code réel (SHA-256 natif, Web Crypto, aucun mock).
  • Gratuit, libre, sans compte, sans tracking.
  • Pensé pour les 15–25 ans curieux, ouvert à tous.
